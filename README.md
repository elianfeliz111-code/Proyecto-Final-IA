# Detector de Enfermedades en Plantas

Aplicacion de Deep Learning para identificar enfermedades en plantas
a partir de fotografias de hojas.

## Descripcion

El usuario sube una imagen de una hoja o parte afectada de la planta
y el sistema utiliza un modelo ResNet-50 entrenado con el dataset
PlantVillage para clasificar la enfermedad presente o determinar
si la planta esta sana.

## Plantas soportadas

- Manzana, Arandano, Cereza
- Maiz, Uva, Naranja
- Durazno, Pimiento, Papa
- Frambuesa, Soya, Calabaza
- Fresa, Tomate
